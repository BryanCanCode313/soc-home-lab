# SOC Analyst Home Lab

## Overview

This repository documents my journey of building a professional Security Operations Center (SOC) home lab from the ground up. The goal of this project is to develop hands-on cybersecurity skills that mirror the responsibilities of a Tier 1 SOC Analyst.

Rather than only studying cybersecurity concepts, I am building, configuring, monitoring, and investigating my own environment using industry-standard tools.

---

## Objectives

- Build a realistic SOC analyst lab
- Learn Windows event logging
- Configure Sysmon for endpoint visibility
- Deploy Splunk Enterprise as a SIEM
- Configure the Splunk Universal Forwarder
- Collect and analyze security logs
- Practice incident detection and triage
- Investigate simulated attacks
- Document findings like a security analyst
